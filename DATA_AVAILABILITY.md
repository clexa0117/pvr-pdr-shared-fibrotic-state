# Data availability

This release does not redistribute raw public datasets or large intermediate count matrices. The analysis uses public datasets and portal records that can be obtained from the original repositories.

## Primary single-cell datasets

- `GSE294329`: PVR discovery cohort.
- `GSE245561`: PDR discovery cohort.
- `GSE165784`: independent PDR cohort with auxiliary PVR sample context.

## External validation datasets

- `GSE179603`: PVR membrane and control membrane tissue validation.
- `GSE60436`: PDR fibrovascular membrane and retina validation.
- `GSE41019`: PVR membrane and retina auxiliary validation.
- `SCP2582`: independent human PVR single-cell object used for cell-context validation.

## Model and public proteomics support

- `GSE228934`: human stem-cell-derived RPE intervention model.
- `GSE274480`: rabbit PVR temporal model.
- `PXD077831`: baseline vitreous proteomics before subsequent PVR development.
- `PXD070155`: recurrence-associated aqueous proteomics context.

## Release boundary

Included files are analysis scripts, selected derived tables, reproducibility manifests, source-data accession records, and release metadata. Excluded files include raw archives, raw matrices, downloaded portal objects, old exploratory workspaces, local tests, generated figure images, figure-only manifests, per-cell annotation exports, full manuscript drafts, author-contact details, manuscript-generation scripts, and submission-only Word/PDF artifacts.
